# RecipeHub Database System

Database Systems | SQL | ER Modeling | DBMS Design

---

## Overview
RecipeHub is a relational database system developed as a university Database Systems course project.

The platform is designed to support a recipe-sharing community that connects cooking enthusiasts, chefs, nutritionists, and companies through an interactive database-driven system.

The project focuses on database design, ER modeling, relational schema creation, normalization, and SQL-based implementation using Visual Studio.

---

## Problem Definition
The system aims to provide a comprehensive platform for:
- Discovering and sharing recipes
- Managing nutritional information
- Creating diet plans
- Promoting food products
- Supporting user engagement through ratings, likes, follows, and collections

The platform supports multiple user roles with different privileges and interactions.

---

## System Users

### Cooking Enthusiasts
- Search recipes using filters
- Save favorite recipes
- Create “To-Cook” lists
- Rate and comment on recipes
- Follow chefs and nutritionists

### Chefs
- Create and manage recipes
- Share recipe collections
- Suggest ingredient substitutes
- Track recipe popularity and engagement

### Companies
- Promote products through recipes
- Collaborate with chefs
- Analyze recipe engagement

### Nutritionists
- Create diet plans
- Add nutritional information
- Publish healthy eating blogs
- Guide users toward dietary goals

### Admin
- Approve or reject chefs, nutritionists, and companies
